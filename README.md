# 🚀 Leonardo - Meu Portfólio Pessoal

Este é o repositório do meu site de portfólio pessoal, onde apresento meus projetos, habilidades e trajetórias na área de desenvolvimento. O objetivo é criar uma experiência visual limpa e direta para quem deseja conhecer meu trabalho.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando tecnologias fundamentais de desenvolvimento web:

* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização, layout (Flexbox/Grid) e animações.
* **JavaScript:** Interatividade (menus, scroll suave, etc.).

---

## 🎨 Funcionalidades

- ✅ **Design Responsivo:** Adaptado para dispositivos móveis, tablets e desktops.
- ✅ **Seção de Projetos:** Galeria com links diretos para repositórios.
- ✅ **Formulário de Contato:** Integração com links diretos.
